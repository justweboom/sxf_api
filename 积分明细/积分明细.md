# 积分详细统计接口

## 接口信息

- **接口路径**: `/api/integral/detail`
- **请求方法**: GET
- **接口描述**: 获取用户积分详细统计信息，包括当前积分、月度收支、今日收支、累计收支及积分记录列表

## 请求参数

| 参数名 | 类型 | 必填 | 默认值 | 描述 |
|-------|------|------|-------|------|
| type | string | 否 | all | 筛选类型：all(全部)、income(收入)、expense(支出)、transfer(转赠) |
| page | int | 否 | 1 | 页码 |
| limit | int | 否 | 20 | 每页条数 |

## 请求头

| 字段名 | 类型 | 必填 | 描述 |
|-------|------|------|------|
| Authorization | string | 是 | Bearer token |

## 字段说明

### 响应数据结构

| 字段名 | 类型 | 描述 |
|-------|------|------|
| current_integral | float | 当前积分余额 |
| monthly | object | 本月统计数据 |
| monthly.income | float | 本月收入积分 |
| monthly.expense | float | 本月支出积分 |
| today | object | 今日统计数据 |
| today.income | float | 今日收入积分 |
| today.expense | float | 今日支出积分 |
| total | object | 累计统计数据 |
| total.income | float | 累计收入积分 |
| total.expense | float | 累计支出积分 |
| list | array | 积分记录列表 |
| count | int | 记录总数 |
| times | array | 时间分组列表 |

### 积分记录项

| 字段名 | 类型 | 描述 |
|-------|------|------|
| id | int | 记录ID |
| uid | int | 用户ID |
| title | string | 记录标题 |
| category | string | 分类（integral: 积分） |
| type | string | 类型 |
| number | float | 积分数量 |
| balance | float | 操作后余额 |
| pm | int | 收支类型（1: 收入, 0: 支出） |
| add_time | string | 添加时间 |
| time_key | string | 时间键（年月） |
| time | string | 时间（年月） |
| day | string | 日期（年月日） |
| is_frozen | int | 是否冻结（1: 是, 0: 否） |

## 示例请求

### 1. 获取全部积分记录

```bash
curl -X GET "http://localhost:8000/api/integral/detail?type=all" \
  -H "Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
```

### 2. 获取收入积分记录

```bash
curl -X GET "http://localhost:8000/api/integral/detail?type=income" \
  -H "Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
```

### 3. 获取支出积分记录

```bash
curl -X GET "http://localhost:8000/api/integral/detail?type=expense" \
  -H "Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
```

### 4. 获取转赠积分记录

```bash
curl -X GET "http://localhost:8000/api/integral/detail?type=transfer" \
  -H "Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
```

## 错误响应

| 错误码 | 错误信息 | 描述 |
|-------|---------|------|
| 410000 | 请登录 | 未登录或token无效 |
| 410001 | 无权限访问 | 权限不足 |

## 实现说明

1. **筛选逻辑**：
   - `all`：查询所有积分记录
   - `income`：只查询收入记录（pm=1）
   - `expense`：只查询支出记录（pm=0）
   - `transfer`：只查询转赠相关记录（type=transfer_in或transfer_out）

2. **统计数据**：
   - 当月统计：从当月1号到当月最后一天
   - 今日统计：从今日0点到23:59:59
   - 累计统计：所有历史记录

3. **分页处理**：
   - 默认每页20条记录
   - 支持通过page和limit参数自定义

4. **时间格式**：
   - add_time：YYYY-MM-DD HH:MM:SS
   - time_key和time：YYYY-MM
   - day：YYYY-MM-DD

5. **冻结状态**：
   - is_frozen=1表示积分处于冻结状态
   - is_frozen=0表示积分正常可用
